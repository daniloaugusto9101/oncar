# React + Tailwindcss + API Node + BD Postgre

## Sobre

Este é um projeto para gestão de veículos, podendo visualizar, ver detalhes e excluir

## Intalação

### Oncar

1. `$ cd oncar`
2. `$ npm install` or `$ yarn install`
3. `$ npm run dev` or `$ yarn run dev`

## Bibliotecas

### Front-end

- `react-router-dom`
- `Axios`
- `react-icons`

### Back-end

- `express`
- `knex`
- `nodemon`
- `cors`
- `pg`
- `dotenv`
